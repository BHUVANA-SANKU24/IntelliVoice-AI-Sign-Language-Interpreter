
#  IntelliVoice – AI Sign Language Interpreter

IntelliVoice is a real-time AI-powered sign language interpreter that enables seamless communication between deaf/mute individuals and others. The system uses computer vision to detect hand gestures and convert them into text and speech, while also supporting text-to-sign animation for two-way interaction.

---

## 🚀 Features

* 📷 **Sign → Text (Real-time Detection)**

  * Detects hand gestures using camera input
  * Converts gestures into letters/words
  * Displays confidence score for predictions

* 🔊 **Text → Speech**

  * Converts detected signs into spoken output

* ✍️ **Text → Sign Animation**

  * Converts typed text into sign language visuals

* 🔄 **Conversation Mode**

  * Enables real-time two-way communication
  * Displays chat-style interaction between users

* 📊 **Confidence Tracking**

  * Visual confidence bar for gesture prediction accuracy

* 🎨 **Modern UI/UX**

  * Dark/light theme
  * Interactive dashboard
  * Smooth animations and accessibility features

---

## 🧠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Computer Vision:** MediaPipe Hands
* **AI/ML:** Pre-trained hand landmark detection models
* **Other:** Canvas API, Web Speech API

---

## ⚙️ How It Works

1. Camera captures live video feed
2. MediaPipe detects hand landmarks
3. Gesture detection logic processes landmarks
4. Recognized signs are converted into text
5. Text can be spoken or converted back into sign animations
